# breakeven.md
## Unit Economics
### Cost per Active User
Based on the infrastructure requirements for idea-luminate, we estimate the following costs:
- Compute: $0.05 per user per month (assuming 10 hours of AI processing time per month at $0.005 per hour)
- Storage: $0.01 per user per month (assuming 100MB of storage per user at $0.0001 per MB)
- Bandwidth: $0.01 per user per month (assuming 100MB of data transfer per user at $0.0001 per MB)
Total cost per active user: $0.07 per month

## Pricing Tiers
We propose the following pricing tiers for idea-luminate:
1. **Spark**: $9/month - access to basic AI-powered idea generation tools, limited to 10 ideas per month
2. **Glow**: $29/month - access to advanced AI-powered idea generation tools, unlimited ideas per month, and priority customer support
3. **Illuminate**: $99/month - access to all features, including personalized idea validation, dedicated customer success manager, and quarterly strategy sessions

## Customer Acquisition Cost (CAC) Range
Based on industry benchmarks, we estimate the CAC range for idea-luminate to be between $15 and $30 per user.

## Lifetime Value (LTV) Estimate
Assuming an average revenue per user (ARPU) of $29 (based on the Glow tier) and a customer lifetime of 12 months, we estimate the LTV to be:
LTV = ARPU x customer lifetime = $29 x 12 = $348
However, considering the tiered pricing structure, we'll use a weighted average ARPU. Assuming 20% of users on Spark, 60% on Glow, and 20% on Illuminate, the weighted average ARPU is:
Weighted average ARPU = (0.2 x $9) + (0.6 x $29) + (0.2 x $99) = $1.8 + $17.4 + $19.8 = $38.99
LTV = weighted average ARPU x customer lifetime = $38.99 x 12 = $467.88

## Break-even Analysis
To calculate the break-even point, we'll use the following formula:
Break-even point = (CAC + cost per active user) / (LTV - cost per active user)
Using the estimated CAC range and LTV, we get:
Break-even point (low CAC): (15 + 0.07) / (467.88 - 0.07) ≈ 16.07 / 467.81 ≈ 0.034 or 3.4% of total users
Break-even point (high CAC): (30 + 0.07) / (467.88 - 0.07) ≈ 30.07 / 467.81 ≈ 0.064 or 6.4% of total users

## Break-even Users Count
Assuming an average cost per active user of $0.07 and using the weighted average ARPU, we can calculate the break-even users count:
Break-even users count (low CAC): 16.07 / 0.034 ≈ 472 users
Break-even users count (high CAC): 30.07 / 0.064 ≈ 469 users

## Path to $10K MRR
To reach $10,000 MRR, we'll need:
- 334 users on the Glow tier ($29 x 334 = $9,706)
- 101 users on the Illuminate tier ($99 x 101 = $9,999)
A combination of both tiers can help us reach the $10K MRR milestone. For example:
- 200 users on the Glow tier ($29 x 200 = $5,800)
- 34 users on the Illuminate tier ($99 x 34 = $3,366)
- 10 users on the Spark tier ($9 x 10 = $90)
Total MRR: $5,800 + $3,366 + $90 = $9,256
This combination is close to the target, and we can adjust the pricing tiers or user acquisition strategies to reach the $10K MRR milestone.
