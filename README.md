# Idea Luminate
Idea Luminate is a system that sends regular updates to users on their progress and ideas, along with relevant market trends and user needs.

## Usage
To use Idea Luminate, create an instance of the `IdeaLuminate` class and add ideas, market trends, and user needs. You can then get updates for specific ideas and schedule updates to be sent at a later time.

## Testing
To run the tests, use the `pytest` command.
